## 🧠 Core Intelligence Layers

### 1. Thinking + Action Layer (Phase 9.5)
- **Structured Planning**: Before answering, the LLM plans its response using an internal "thought" and "intent" process.
- **Tool Selection**: Automatically chooses and executes tools (e.g., memory retrieval, Python execution) based on user intent.
- **JSON Enforcement**: Uses strict JSON output to ensure reliable tool-calling and response formatting.

### 2. RETINA (Visual Intelligence)
- **Smart Move**: Automatic model switching when an image is detected.
- **Confidence Scoring**: Decides between **Retrieve** (from memory), **Generate** (new image), or **Hybrid** (using reference).
- **Match Reference**: Uses ComfyUI IPAdapter to maintain visual consistency (e.g., same character or style) when generating variations.

### 3. CHRONOS (Episodic Memory)
- **Long-term Recall**: Multi-session episodic memory that indexes and retrieves past conversation highlights.
- **Shadow Memory**: Allows users to "pin" specific terminal logs or facts directly into the AI's current context.
